# Syntrixa Memory Skill

Use this skill when working on:

- Working Memory
- Long Memory
- Private Memory
- Shared Memory
- Identity Context Retrieval
- memory indexing
- memory promotion rules
- agent-specific memory boundaries

## Rules

Memory must support the Hybrid Adaptive Memory Model.

Memory retrieval should provide context to the supervisor and specialist agents without allowing agents to bypass execution boundaries.

Do not store secrets unless explicitly required and approved.

## Memory Types

- Working Memory: active interaction context.
- Long Memory: durable cross-session facts and decisions.
- Private Memory: agent- or identity-specific memory.
- Shared Memory: cross-agent operational context.
- Identity Context Retrieval: retrieval of relevant user/client/account context before reasoning.
